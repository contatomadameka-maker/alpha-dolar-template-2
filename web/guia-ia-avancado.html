Visão Geral

O que é o Modo IA Avançado?

O Modo IA Avançado é o cérebro do Alpha Dolar 2.0. Ele usa Machine Learning em tempo real para analisar os tiques do mercado e só liberar entradas quando o score de confiança é favorável.

📐

Análise de Frequência

Analisa quais dígitos (0–9) aparecem mais nos últimos 50 tiques. Identifica desvios estatísticos antes de qualquer entrada.

⚖️

Ratio Par/Ímpar

Calcula proporção entre dígitos pares e ímpares em tempo real. Desequilíbrio acima do esperado gera sinal de entrada.

🔗

Sequência Atual

Detecta sequências de 3+ resultados iguais. Longa sequência ativa estratégia de reversão automaticamente.

🎮

Perdas Virtuais

Simula N perdas antes de usar capital real. O bot só entra com dinheiro após confirmar o padrão virtualmente.

🔢

Score de Entrada

Score mínimo de 58% exigido para liberar qualquer entrada. Sem sinal forte, o bot simplesmente aguarda.

⚡

WebSocket em Tempo Real

Conectado diretamente à API Deriv. Cada tique é analisado em milissegundos — nunca perde um movimento.

✨ Como ativar: No dashboard, ative o toggle "Avançado" no canto superior do painel esquerdo. O painel muda para o modo IA com todos os campos avançados disponíveis. Depois configure e clique em Iniciar Robô IA.

Machine Learning

Como o Motor IA Aprende

O Motor IA Alpha opera em duas fases distintas. Entender isso é essencial para saber o que esperar nos primeiros minutos:

Fase 1
COLETA · Primeiros ~5 min

Coletando Dados

O bot precisa de pelo menos 150 amostras para treinar o modelo. Durante esse período, ele usa análise estatística básica (fallback) com score mínimo de 60%.

Você verá no painel: Coletando (X/150)

Fase 2
ML ATIVO · Após 150 amostras

RandomForest Treinado

O modelo RandomForest entra em ação com 80 árvores de decisão. Retreina automaticamente a cada 50 novos tiques, ficando cada vez mais preciso.

Você verá no painel: ML ativo · acurácia: XX%

💡 Dica: Deixe o bot rodar por pelo menos 10 minutos antes de avaliar o desempenho. Os primeiros trades usam análise estatística básica — o ML fica mais preciso conforme aprende o comportamento do mercado atual.

MERCADO

Mercado + Tipo de Negociação

Escolha o índice sintético e o tipo de contrato (DIGITEVEN, DIGITODD, DIGITOVER, DIGITUNDER). O Motor IA se calibra especificamente para o tipo escolhido.

DURAÇÃO

Duração e Unidade

Para dígitos use sempre 1 Tique. Isso garante execução máxima de trades e mais dados para o ML aprender. Durações maiores reduzem a frequência.

MULTIPLICADOR

Multiplicador de Martingale

Define quanto o stake aumenta após uma perda. 2.2× é o equilíbrio ideal — recupera a perda anterior + dá lucro. Nunca use acima de 3.0× sem banca alta.

VIRTUAIS

Perdas Virtuais

O bot simula N perdas antes de entrar com dinheiro real. Garante que só entre quando o padrão está confirmado. 3 perdas virtuais é o padrão recomendado.

NEG. VIRTUAIS

Quando Usar Virtuais

Na perda: só simula quando há perda acumulada (recomendado). Sempre: simula antes de toda entrada. Nunca: entra direto sem simulação.

FINANCEIRO

Quantia · Lucro Alvo · Stop Loss

Stake inicial por trade. Lucro alvo encerra o bot ao atingir. Limite de perda para o bot antes de perder mais. Defina com base na sua banca.

3
Estratégias

Qual Tipo de Contrato Usar?

O Motor IA se adapta ao tipo de contrato escolhido. Cada um tem uma estratégia de análise diferente:

DIGITEVEN
Par (Even)

O último dígito do preço deve ser par (0, 2, 4, 6, 8). O ML calcula P(par) e entra quando ≥ 58%.

✅ Ideal quando: Pills pares dominando (>55%)
DIGITODD
Ímpar (Odd)

O último dígito deve ser ímpar (1, 3, 5, 7, 9). O ML calcula P(ímpar) e entra quando ≥ 58%.

✅ Ideal quando: Pills ímpares dominando (>55%)
DIGITOVER
Superior (Over 4)

O último dígito deve ser maior que 4 (5, 6, 7, 8, 9). Entra quando os altos estão abaixo de 40% (reversão esperada).

✅ Ideal quando: Dígitos altos frios no momento
DIGITUNDER
Inferior (Under 5)

O último dígito deve ser menor que 5 (0, 1, 2, 3, 4). Entra quando os baixos estão abaixo de 40% (reversão esperada).

✅ Ideal quando: Dígitos baixos frios no momento

Dicas Avançadas

Como Extrair o Máximo do Motor IA

📊

Leia os Pills Antes de Iniciar

Vá na aba Dígitos antes de ligar o bot. Se os pills mostram 60%+ pares → use DIGITEVEN. 60%+ ímpares → DIGITODD. Isso amplifica o score do ML.

⏰

Deixe o ML Aquecer

Os primeiros 5–10 minutos são de aprendizado. Não julgue o desempenho no início. O bot fica progressivamente mais preciso conforme coleta dados do mercado atual.

🔄

Troque o Tipo se Necessário

Se o mercado mudou de comportamento (pills passaram de par para ímpar), pare o bot, mude o tipo e reinicie. O ML se calibra para o novo tipo.

🛑

Respeite o Stop Loss

Configure um limite de perda real e não mude durante a sessão. Nenhum ML é 100% — o stop loss protege sua banca em dias de mercado adverso.

🌙

Cuidado com Horários

Mercados sintéticos funcionam 24h, mas costumam ter padrões melhores em certos horários. Observe em qual período o bot performa melhor para você.

📉

Sequência de 5+ Perdas?

Pare o bot. O mercado provavelmente mudou de comportamento. Analise os pills, aguarde 5 minutos e reinicie com um novo ciclo de aprendizado.

💡 Combinação poderosa: Use o Modo IA Avançado no painel esquerdo para operar automaticamente, e ao mesmo tempo acompanhe os pills na aba Dígitos para detectar mudanças de padrão cedo. Quando os pills mudarem, você ajusta o tipo do bot antes de começar a perder.
